# HotHAIR - Сайт для парикмахерской

Современный, стильный сайт для парикмахерской HotHAIR с темным дизайном, LED-подсветкой и элементами дерева, мха и стали.

## Функциональность

- Адаптивный дизайн (мобильные устройства, планшеты, десктоп)
- Анимированный логотип и элементы интерфейса
- Форма записи на услуги с отправкой данных
- Интеграция с мессенджерами (Telegram, WhatsApp)
- Галерея работ
- Информация о мастере
- Контактная информация

## Технологии

- HTML5, CSS3
- JavaScript (чистый JS)
- GSAP для продвинутых анимаций
- Intersection Observer API для анимаций при прокрутке
- Временное хранение данных в localStorage (для демонстрации)

## Настройка отправки данных формы

Для настройки отправки данных формы на email рекомендуется использовать сервис EmailJS:

1. Зарегистрируйтесь на [emailjs.com](https://www.emailjs.com/)
2. Создайте сервис (например, Gmail)
3. Создайте шаблон письма с переменными для данных формы
4. Подключите библиотеку EmailJS, добавив следующий скрипт в index.html:
   ```html
   <script src="https://cdn.emailjs.com/sdk/2.3.2/email.min.js"></script>
   ```
5. Инициализируйте EmailJS с вашим ID пользователя в script.js:
   ```javascript
   emailjs.init("YOUR_USER_ID");
   ```
6. Раскомментируйте и настройте функцию sendEmail() в script.js, заменив параметры на свои:
   ```javascript
   emailjs.send('service_id', 'template_id', data, 'user_id')
   ```

## Настройка ссылок мессенджеров

Для настройки ссылок на мессенджеры необходимо заменить:

1. В файле index.html найдите ссылки на Telegram и WhatsApp:
   ```html
   <a href="https://t.me/hothair_salon" class="social-btn telegram" target="_blank">
   <a href="https://wa.me/79991234567" class="social-btn whatsapp" target="_blank">
   ```

2. Замените `hothair_salon` на ваш username в Telegram
3. Замените `79991234567` на ваш номер телефона в формате WhatsApp

## Административная панель

Для просмотра записей через форму можно использовать административную панель (admin.html).
Панель показывает записи, сохраненные в localStorage (в демонстрационных целях).

Для реального проекта рекомендуется настроить полноценную серверную часть с базой данных и авторизацией.
